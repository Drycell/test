# wormwing

Structure-first evolution research scaffold for a connectome-derived CTRNN controlling a 3D winged worm body.

## Quickstart

```bash
make install
make smoke
make smoke-exp001
```

## Main experiment

```bash
make train-exp001
make eval-latest
make report-latest
```
